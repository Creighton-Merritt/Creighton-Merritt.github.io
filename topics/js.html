Loops

In Javascript, a loop is a feature that faciliatates the execution of a set of commands or functions repeatedly while the condition evaluates to true. There are several different types of loops.

For Loop: loops through a block of code a specified number of times

for (i = 0; i < 5; i++) {
  console.log(i);
}

While Loop: loops through a block of code while a specified condition is true

var i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Do/While Loop: loops through a block of code once, then repeates the loop if the specified condition is still true

var i = 0;
do {
  console.log(i);
  i++;
}
while (i < 5)

Output for all 3 loops

console:
0
1
2
3
4

Conditional Statements: If, Else, Else If

The 'If' statement can be used if you want to check one specific condition

if (condition) {
 // code to be executed if condition is true;
}

The 'If/Else' statement can be used if you have to check two conditions and execute different set of codes

if (condition) {
 // code to be executed if condition is true;
}
else {
 // code to be executed if condition is false;
}

The 'If/Else If/Else' statement can be used if you want to check more than two conditions

if (condition1) {
 // code to be executed if condition1 is true;
}
else if (condition2) {
 // code to be executed if condition2 is true;
}
else {
 // code to be executed if both conditions are false
}

Functions & Parameters

A Javascript function is a block of code designed to perform a particular task. It is executed when something else calls it. It is defined with the function keyword, followed by a name and parenthese (). If the function has parameters they should be separated by commas.

function name(parameter1, paremeter2) {
 // code to be executed
}

Variables

Javascript variables are containers for storing data values. Unlike c++, variables do not need to declare a type such as int, string, char, etc. Rather, a variable is declared with the keyword var.

var myName = "Merritt";
var myAge = 39;

Arrays and Associative Arrays

An array is a single variable that is used to store multiple elements. We can store different types of elements in a single array. The elements in an array are accessing using [].

var cars = [];
// initialize while declaring
var cars = ["Subaru", "Mitsubishi", "Scion", 3];
// initialize after declaring
cars[0] = "Subaru";
cars[1] = "Mitsubishi";
cars[2] = "Scion";
cars[3] = 3;

Associative arrays are really Javascript objects. We can create an associative array using Object().

var obj = new Object();
  obj["one"] = 1;
  obj["two"] = 2;
  obj["three"] = 3;
for(var i in obj)
{
  console.log(obj[i]);
}
