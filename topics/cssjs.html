Manipulating CSS Class Property Using JavaScript

Changing CSS using JavaScript

We can use JavaScript to change the CSS properties by using the HTML DOM to change the style of HTML elements. Many different style properties can be changed in this way including color, font family, alignment of elements, and background color. You can find a full list of HTML DOM style objects at W3Schools

It is done by using this syntax:

document.getElementById(id).style.property = new style


Here you can change the font size of this paragraph element by clicking a button:

I want to change the size of this sentence


The code for this example is as follows-
HTML:

<p id=“cSize”>I want to change the size of this sentence</p>
<button type=“button” onclick=“changeSize()”>Change My Size</button>


Javascript:

function changeSize() {
 var cSize = document.getElementById("cSize").value;
 document.getElementById("cSize").style.fontSize = "30px";
}
