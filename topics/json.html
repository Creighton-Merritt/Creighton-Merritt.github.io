JSON Parse & Stringify

Objects

JSON stands for "Javascript Object Notation". It is used for storing and sending data from a server to a web page. JSON files can contain objects or arrays. Objects are defined with curly braces and arrays are defined with square brackets.

Parsing JSON

Json is used to exchange data between the server and the user. The JSON data received from the server is in string format, and needs to be parsed to become a JavaScript object. This can be done by using the JSON.parse() method.

Let's say that we want to get info on a student. We query the server and the server returns this JSON string:

'{ "name":"Merritt", "age":22, "city":"Salt Lake City", "degree":"Software Engineering"}'

Now use the JSON.parse() method to convert the text into a JavaScript object is:

var studentObj = JSON.parse('{ "name":"Merritt", "age":22, "city":"Salt Lake City", "degree":"Software Engineering"}');

The whole string is now a JavaScript object. We can access the values with the . notation

console.log(studentObj.name);

// Output Merritt

Stringify

The JSON.stringify() method will convert a JavaScript object into a string. If we wanted to change our new studentobj back into a string we would do this:

var studentDataString = JSON.stringify(studentObj);
console.log(studentDataString);

// Output '{ "name":"Merritt", "age":22, "city":"Salt Lake City", "degree":"Software Engineering"}'
