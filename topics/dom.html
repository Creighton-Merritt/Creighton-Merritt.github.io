Document Object Model

DOM Manipulation

DOM stands for Document Object Model. When an html page is loaded, the browser creates a Document Object Model of the page. It includes all of the html tags and properties.

JavaScript has the power to add, modify, and remove HTML elements and attributes, change CSS styles, and create new HTML events in the page.
For this example I'm going to focus on createElement, insertBefore, and appendChild.
