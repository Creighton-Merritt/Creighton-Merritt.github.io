Web Storage

Web storage allows web applications to store data locally within the user's browser. Before HTML5, data had to be stored in cookies that were included in every server request. Web storage can be stored locally, so it is more secure and it does not affect website performance.

  • There are two different kinds of web storage: local storage, and session storage.
  • Session storage will store the data for one session (until the browser tab is closed.)
  • Local storage data has no expiration date and will persist until it is deleted manually.


Syntax for saving data in key-value pairs:

localStorage.setItem("key", "value");

localStorage.setItem("firstname", "Merritt");


Syntax for saving data as objects:

localStorage.setItem("objects", JSON.stringify(objects));

let students = [
 new Student('Merritt', 'Creighton', '999'),
 new Student('Merritt', 'Tuley', '999')
];
localStorage.setItem("students", JSON.stringify(students));


Syntax for reading data from local storage:

var getKey = localStorage.getItem("key");


Syntax for removing data from local storage:

localStorage.removeItem("key");
