JavaScript Events

HTML Events

An HTML event is something that happens to HTML elements, such as an input field being changed, a button being clicked, a page loading, or a mouse hovering over an element. There are many different ways to trigger an event. JavaScript can be used to react to HTML events and execute code when events are detected.

onclick

The onlick event is commonly used for JavaScript to run a function when the user clicks on a button

HTML:
<button type=“button” id=“btn” onclick=“changeButton()”>Click Me</button>

JavaScript:
function changeButton() {
 var text = document.getElementById("btn");
 if (text.innerHTML === "Click Me") {
  text.innerHTML = "Clicked!";
 } else {
  text.innerHTML = "Click Me";
 }
}



onload

The onload event will execute JavaScript immediately after a page has been loaded. The event can be used to check the visitor's browser type and browser version, and load the proper version of a web page based on the information. It can also be used to check if cookies are enabled or if cookies already exist for the user.


touchstart

The touchstart event is used for touch screens only i.e. mobile phones and tablets. It executes JavaScript when the user touches a P element.

HTML:
<p> name=“pBold” ontouchstart=“changeFont()”>Touch here to make me bold</p>

JavaScript:

Touch here to make me Bold
